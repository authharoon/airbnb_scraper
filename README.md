# Airbnb Scraper

A Python-based scraper for extracting Airbnb listing information.

## Features

- Listing Title
- Listing URL
- Bedrooms
- Beds
- Bathrooms
- Maximum Guests
- Price
- Rating
- Number of Reviews
- Address
- City
- Country
- Latitude
- Longitude



## Technologies

- Python
- Requests
- Parsel
- Pandas

## Files

- `airbnb_scraper.py` – Main scraper
- `Airbnb Sample.xlsx` – Sample extracted dataset

## Output

The scraper exports structured Airbnb listing data to Excel format .
